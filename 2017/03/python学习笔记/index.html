python学习笔记

数据结构

  • 列表（list）
  • 元组（tuple）
  • 字典（dist）
  • SET

函数参数类型

  • 默认参数
  • 可变参数
  • 关键字参数
  • 尾递归（尾递归是指，在函数返回的时候，调用自身本身，并且，return语句不能包含表达式）

高级语法

  • 切片（slice）
  • 迭代（Iteration、enumerate）
  • 列表生成（range … ）
  • 生成器（generator）

高阶函数 (把函数作为参数传入，这样的函数称为高阶函数)

  • map/reduce
  • filter
  • sorted
  • 函数返回值
  • 闭包 （返回函数不要引用任何循环变量，或者后续会发生变化的变量）
  • 匿名函数
  • Decorator（装饰器）
  • functools（偏函数partial）
